# Personal Portfolio Website

## Overview

This is my personal portfolio website built with React, showcasing my projects, technical skills, and professional journey as I transition from Industrial Engineering into Data Engineering.

The goal of this project is to serve as a central hub for my work, documenting my learning process, portfolio projects, and continuous growth in data engineering, analytics, automation, and software development.

As I continue building projects and expanding my technical skills, this portfolio will evolve alongside my career.
